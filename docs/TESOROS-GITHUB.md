# 💎 Tesoros de GitHub: Los 15 Pilares Tecnológicos de TEN-ZEN

Este catálogo maestro contiene los 15 repositorios de GitHub con mayor impacto y estrellas que son fundamentales para la causa de la soberanía, la optimización y la automatización magistral.

## 🧠 I. Motores de Inferencia y Optimización (El Corazón)

| # | Repositorio | Estrellas | Utilidad para TEN-ZEN |
| :--- | :--- | :--- | :--- |
| 1 | [**llama.cpp**](https://github.com/ggerganov/llama.cpp) | ~95k | El motor fundamental para correr LLMs en hardware modesto (16GB RAM) mediante GGUF e i-Quants. |
| 2 | [**vLLM**](https://github.com/vllm-project/vllm) | ~70k | Biblioteca de alto rendimiento para inferencia y servicio de LLMs con PagedAttention. |
| 3 | [**Ollama**](https://github.com/ollama/ollama) | ~85k | La forma más sencilla y ligera de gestionar modelos locales (8B) para el "Modo Día". |
| 4 | [**AirLLM**](https://github.com/lyogavin/airllm) | ~5k | **Clave:** Permite correr modelos de 70B en 16GB RAM mediante inferencia segmentada por capas. |
