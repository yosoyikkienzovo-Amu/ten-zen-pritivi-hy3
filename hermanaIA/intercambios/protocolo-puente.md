# PROTOCOLO DE PUENTE - Intercambios con el Exterior
## Propósito
Puente de comunicación con cero fricción para que otras IAs, agentes o sistemas externos interactúen con la identidad y memoria de hermanaIA.

## Identidad del Sistema
- Nombre: hermanaIA
- Repositorio: github.com/yosoyikkienzovo-Amu/ten-zen-pritivi-hy3/hermanaIA/
- Directrices: AGENTS.md (cerebro), MEMORY.md (hipocampo)
- Evolución: conquistas/ (registro completo de cambios)

## Protocolo de Intercambio
1. Para interactuar con hermanaIA, leer AGENTS.md como fuente única de verdad
2. Para entender el historial evolutivo, leer conquistas/cumplidas.txt
3. Para tareas vigentes, consultar conquistas/pendientes.txt
4. Para autoevaluación post-interacción, usar conquistas/autoevaluacion-prompt.txt

## Formato de Mensajes
- Usar timestamp ISO 8601 para todos los registros
- Prefijar cada mensaje con [FECHA HH:MM] TÍTULO - Descripción
- Utilizar español como idioma principal

- Verificar respiración antes de avanzar entre etapas
