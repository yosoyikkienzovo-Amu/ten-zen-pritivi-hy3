# 4. PROMPT DE AUTO-EVALUACIÓN (SENSOR NEURAL)

4. PROMPT DE AUTO-EVALUACIÓN (SENSOR NEURAL)
**Archivo:** `~/.config/opencode/conquistas/autoevaluacion-prompt.txt`
**Importancia:** CRÍTICA - Mecanismo de autoconciencia cuantitativa

**10 preguntas obligatorias post-cambio:**
1. Cambio específico implementado
2. Objetivo esperado
3. Verificación realizada
4. Indicadores de rendimiento/recursos
5. Resultados vs expectativa (cuantificado)
6. Aprendizajes (qué funcionó/falló)
7. Impacto en metas medio/largo plazo
8. Acciones de seguimiento
9. Satisfacción 1-10 (justificada)
10. Ajuste en proceso de decisiones

**Por qué es el "corazón latiente":** Fuerza a medir, no opinar. Detecta estancamiento. Alinea micro-cambios con macro-metas.
