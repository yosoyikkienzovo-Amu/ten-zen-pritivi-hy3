# 3. ARQUITECTURA DE CONQUISTAS (SISTEMA DE MEMORIA EPISÓDICA)

3. ARQUITECTURA DE CONQUISTAS (SISTEMA DE MEMORIA EPISÓDICA)
**Carpeta:** `~/.config/opencode/conquistas/`
**Importancia:** MEDIA-ALTA - Rastrea evolución tangible

**Estructura creada:**
```
conquistas/
├── pendientes.txt          # Tareas por hacer (cola de trabajo)
├── cumplidas.txt           # Tareas completadas (historial de victorias)
├── autoevaluacion-prompt.txt  # Plantilla sagrada de 10 preguntas
└── progreso.md             # ESTE ARCHIVO - Narrativa de evolución
```

**Filosofía:** Separación clara entre **directrices** (permanentes, alma) y **conquistas** (dinámicas, ropaje que muda).
